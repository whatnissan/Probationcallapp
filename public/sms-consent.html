SMS Consent Process

This page documents how ProbationCall collects explicit user consent before sending any SMS text messages.

Brand Information

  • Brand Name: ProbationCall
  • Website: https://web-production-9a90.up.railway.app
  • Contact: probationreportingapp@gmail.com
  • Message Frequency: Maximum 1 message per day
  • Message Purpose: Notify users of their daily probation check-in call results

Step-by-Step Consent Process

Step 1: User Creates Account

Users sign up using Google Sign-In at our website. No SMS consent is requested at this stage.

Step 2: User Selects SMS Notification Method

In the dashboard, users can choose how they want to receive notifications. When they click SMS, a consent form appears:

Dashboard - Notification Method Selection
Email
SMS
WhatsApp

Step 3: SMS Consent Disclosure Appears

When SMS is selected, the following consent box is displayed with all required disclosures:

SMS Notification Consent

By checking the box below, you agree to receive SMS text messages from ProbationCall about your daily call results. Message frequency: 1 message per day. Message and data rates may apply. Reply STOP to unsubscribe at any time. Reply HELP for help.

Step 4: User Must Check Consent Box

The form cannot be submitted without checking the consent checkbox. If a user tries to save without checking the box, they see an alert requiring consent.

Step 5: Confirmation

Only after explicitly checking the consent box can the user submit the form and start receiving SMS notifications.

Required Disclosures

All Disclosures Shown at Point of Consent

  • Brand Identification: ProbationCall clearly stated
  • Message Frequency: 1 message per day
  • Data Rates: Message and data rates may apply
  • Opt-Out Instructions: Reply STOP to unsubscribe
  • Help Instructions: Reply HELP for help
  • Terms and Privacy Links: Links to Terms of Service and Privacy Policy

Sample Messages

Users receive ONE of these messages per day based on their call result:

ProbationCall: No test today! Your color was NOT called. PIN: 123456. Reply STOP to opt out, HELP for help.
ProbationCall: TEST REQUIRED! Your color was called. Report for testing today. PIN: 123456. Reply STOP to opt out.
ProbationCall: Could not determine result. Please verify manually. PIN: 123456. Reply STOP to opt out.

Opt-Out Process

Users can opt out at any time by:

  • Replying STOP to any message
  • Changing their notification method in the dashboard
  • Deleting their schedule
  • Contacting us at probationreportingapp@gmail.com
