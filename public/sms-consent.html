2. Consent Collection Method

Users provide explicit opt-in consent through our website. Consent is collected via a clearly labeled checkbox that users must actively check before they can receive SMS messages.

Opt-In Methods:

  1. Primary Method - Web Form: Users visit our SMS sign-up page at https://web-production-9a90.up.railway.app/sms-signup.html and complete the opt-in form with required checkbox consent.
  2. Secondary Method - Dashboard: Logged-in users can also enable SMS in their dashboard settings, which displays the same consent disclosures and checkbox.

6. Opt-Out Process

Users can opt out at any time using any of these methods:

  • Reply STOP to any message (immediate unsubscribe)
  • Reply CANCEL, END, QUIT, or UNSUBSCRIBE
  • Change notification settings in their dashboard
  • Email probationreportingapp@gmail.com to request removal
